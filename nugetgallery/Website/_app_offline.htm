Some more information:

What is the Chocolatey Gallery

You can develop your own package and share it via the Chocolatey Gallery. Read the documentation for more details on how to create and publish a package. If you don’t plan on submitting a package, there’s no need to register.

What is Chocolatey?

Chocolatey is like apt-get, but for Windows.

How Does Chocolatey Work?

When you use Chocolatey to install a package, it copies the package to your system, runs powershell against chocolateyInstall.ps1 (if there is on in the package) and puts executables on the path (if there are any in the package).
